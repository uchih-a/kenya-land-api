Technical Specification v2.4
Model Architecture

The Thamani ensemble combines three complementary neural architectures: a standard MLP for non-linear feature interactions, a Self-Normalizing Network (SNN) for stable deep representations, and a TabNet attention model for interpretable feature selection. Predictions are fused via a Ridge meta-learner.

Data Pipeline
Scraping & Ingestion
Automated crawlers targeting BuyRentKenya and Property24. Deduplication of 15,000+ records.
Feature Engineering
Haversine distance calculations to CBDs and transport hubs.
Embedding Generation
Latent representations of Nairobi counties using K-Means clustering.
Model Inference
Real-time prediction through the Ridge Meta-Learner API at kenya-land-valuation-project.onrender.com

Known Limitations
Data Coverage
MEDIUM

Dataset limited to 813 high-quality records. Under-represented in rural Machakos and Kajiado regions.

County Classification
RESOLVED

Historical misclassification of Nairobi sub-regions corrected via reverse geocoding on Lat/Lon pairs.

Price Type
NOTE

Model predicts Asking Price. Final transaction prices may vary 5–12% based on negotiation data gaps.
